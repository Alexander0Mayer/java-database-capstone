Admin User Stories:
A1 As an administrator I want to access a list of doctors at specific clinics so that I can keep track of the available ressources.
A2 As an administrator I want to access the appointment schedules of specific doctors so that I can keep track of the available ressources.
A3 As an administrator I want to see the appointments of specific patients so that I can assess whether patients are retained by the clinics.
Patient User Stories:
P1 As a patient I want to access an appointment page that allows me to chose appointments with a doctor of my choice.
P2 As a patient I want to access an overview of all of my upcoming appointments so that I can keep track of them.
P3 As a patient I want to access an overview of my prescriptions so that I can take my medicine as prescribed.
Doctor User Stories:
D1 As a doctor I want to access the patient files so that I can prepare for appointments.
D2 As a doctor I want to access the presciptions handed out to a patient so that I can keep track of their medical history.
D3 As a doctor I want to have a schedule of upcoming appointments with patients so that I can prepare for these.
